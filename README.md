# RaspberryPi-NTP-server

- Develop stratum 1: NPT server with GPS module

### BOM:  
  - Raspberry PI (Zero-W)  
  - GPS module : GYSFDMAXB/AE-GPS  

### Prerequisite  

### Structure  
- GPS with PPS: PPS signal source
- gpsd : GPS signal corrector & provider by IP
- ntpd (or chrony) : NTP server/client
